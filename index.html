Sistema de Gestión de Voluntarios

Plataforma de organización y seguimiento del voluntariado

Este sistema permite organizar la información del voluntariado, gestionar capacitaciones, registrar actividades operativas y facilitar la identificación de cada miembro mediante credenciales con código QR.

Estadísticas del Sistema

Las estadísticas muestran el estado actual del voluntariado dentro del sistema. Los datos se actualizan automáticamente cuando se registran nuevos miembros, capacitaciones o actividades.
